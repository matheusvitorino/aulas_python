# Exercícios de Python

Este repositório reúne exercícios de lógica de programação que estou desenvolvendo durante meus estudos em Python, com o objetivo de fortalecer minha base e evoluir no raciocínio lógico.

## Conteúdos praticados

* Operações matemáticas
* Estruturas condicionais (if / elif / else)
* Estruturas de repetição (while)
* Conversões de valores
* Entrada e saída de dados
* Validações e regras de negócio
* Cálculo de média, salário e consumo
* Comparação de valores
* Verificações lógicas aplicadas ao dia a dia
