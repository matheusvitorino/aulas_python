# Exercícios de Python

Este repositório reúne exercícios de **lógica de programação em Python** desenvolvidos durante meus estudos.
O objetivo é praticar conceitos básicos da linguagem e fortalecer o raciocínio lógico na resolução de problemas.

## Conteúdos praticados

* Operações matemáticas
* Estruturas condicionais (`if` / `else`)
* Conversões de unidades e valores
* Cálculo de média e salário
* Verificação de idade e regras de votação
* Comparação entre números
* Cálculo de Índice de Massa Corporal (IMC)

## Exercícios

* **01-soma.py** → soma de dois números
* **02-desconto.py** → cálculo de desconto em um produto
* **03-celsius.py** → conversão de Celsius para Fahrenheit
* **04-dollar.py** → conversão de dólar para real
* **05-idade.py** → cálculo de idade a partir do ano de nascimento
* **06-habilitação.py** → verificação se a pessoa pode tirar habilitação
* **07-centimetros.py** → conversão de metros para centímetros
* **08-media.py** → cálculo da média de notas
* **09-salario.py** → cálculo de salário com base em horas trabalhadas
* **10-numero.py** → verificação de propriedades de um número
* **11-par.py** → verificação se um número é par ou ímpar
* **12-salariocomregra.py** → cálculo de salário com regras específicas
* **13-voto.py** → verificação do tipo de voto (obrigatório, opcional ou não permitido)
* **14-maioridade.py** → verificação de maioridade
* **15-horas.py** → cálculo de horas trabalhadas
* **16-compararnumeros.py** → comparação entre dois números
* **17-imc.py** → cálculo do Índice de Massa Corporal (IMC)
