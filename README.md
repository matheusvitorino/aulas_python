# Exercícios de Python

Este repositório reúne exercícios de lógica de programação que estou fazendo durante meus estudos em Python.

## Conteúdos praticados

* Operações matemáticas
* Estruturas condicionais (if / else)
* Conversões de valores
* Cálculo de média e salário
* Verificação de idade e voto
* Comparação de números
* Cálculo de IMC

## Exercícios

* **01-soma.py** → soma de dois números
* **02-desconto.py** → cálculo de desconto
* **03-celsius.py** → conversão de Celsius para Fahrenheit
* **04-dollar.py** → conversão de dólar para real
* **05-idade.py** → cálculo de idade
* **06-habilitação.py** → verificação para tirar habilitação
* **07-centimetros.py** → conversão de metros para centímetros
* **08-media.py** → cálculo de média de notas
* **09-salario.py** → cálculo de salário por horas
* **10-numero.py** → verificação de número
* **11-par.py** → número par ou ímpar
* **12-salariocomregra.py** → salário com regras
* **13-voto.py** → verificação do tipo de voto
* **14-maioridade.py** → verificação de maioridade
* **15-horas.py** → cálculo de horas trabalhadas
* **16-compararnumeros.py** → comparação entre números
* **17-imc.py** → cálculo de IMC
