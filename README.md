# Exercícios de Python

Este repositório contém exercícios de lógica de programação que estou desenvolvendo durante meus estudos em Python.

## Conteúdos praticados

- Operações matemáticas
- Estruturas condicionais (if / else)
- Conversões de temperatura
- Cálculo de IMC
- Verificação de idade e voto
- Comparação de números
- Cálculo de salário

## Exercícios

01-soma.py → programa que soma dois números
02-desconto.py → cálculo de desconto de produto
03-celsius.py → conversão de Celsius para Fahrenheit
04-dollar.py → conversão de reais para dólares
05-idade.py → cálculo da idade com base no ano de nascimento
06-habilitação.py → verificação se a pessoa pode tirar habilitação
07-centimetros.py → conversão de metros para centímetros
08-media.py → cálculo da média de notas
09-salario.py → cálculo de salário baseado em horas trabalhadas
10-numero.py → verificação de características de um número
11-par.py → verificação se um número é par ou ímpar
12-salariocomregra.py → cálculo de salário aplicando regras específicas
13-voto.py → verificação se o voto é obrigatório, opcional ou não permitido
14-maioridade.py → verificação se a pessoa é maior ou menor de idade
15-horas.py → cálculo de horas trabalhadas
16-compararnumeros.py → comparação entre dois números
17-imc.py → cálculo do índice de massa corporal (IMC)
